# Feign 无参构造方法问题演示项目 - 整合说明

## 项目定位

本项目 `feign-noargs-demo` 已整合为 `springboot` 父项目的一个子模块。

## 项目结构

```
springboot/                                    # 父项目
├── pom.xml                                    # 父 POM（已添加 feign-noargs-demo 模块）
├── base-common/
├── springboot-database-learning/
├── springboot-redis-learning/
├── ...
└── feign-noargs-demo/                         # 本项目（子模块）
    ├── pom.xml                                # 模块父 POM
    ├── provider-service/                      # 服务提供者
    │   ├── pom.xml                            # 继承 feign-noargs-demo
    │   └── src/
    ├── consumer-service/                      # 服务消费者
    │   ├── pom.xml                            # 继承 feign-noargs-demo
    │   └── src/
    ├── README.md
    ├── START_HERE.md
    ├── 启动脚本.sh
    ├── 停止脚本.sh
    └── 快速测试.sh
```

## Maven 依赖关系

```
springboot (父项目)
    ↓ parent
feign-noargs-demo (模块父 POM)
    ↓ parent
    ├── provider-service
    └── consumer-service
```

### 父项目 POM 配置

`springboot/pom.xml` 中添加了模块：

```xml
<modules>
    <module>base-common</module>
    <module>springboot-circuit-breaker</module>
    <!-- ... 其他模块 ... -->
    <module>feign-noargs-demo</module>  <!-- 新增 -->
</modules>
```

### 模块父 POM 配置

`feign-noargs-demo/pom.xml`：

```xml
<parent>
    <groupId>com.huabin</groupId>
    <artifactId>springboot</artifactId>
    <version>1.0-SNAPSHOT</version>
</parent>

<artifactId>feign-noargs-demo</artifactId>
<packaging>pom</packaging>

<modules>
    <module>provider-service</module>
    <module>consumer-service</module>
</modules>
```

### 子服务 POM 配置

`provider-service/pom.xml` 和 `consumer-service/pom.xml`：

```xml
<parent>
    <groupId>com.huabin</groupId>
    <artifactId>feign-noargs-demo</artifactId>
    <version>1.0.0</version>
</parent>
```

## 编译方式

### 方式 1: 从父项目编译

```bash
# 在 springboot 父项目根目录
cd /path/to/springboot

# 编译整个父项目（包括 feign-noargs-demo）
mvn clean install

# 只编译 feign-noargs-demo 模块及其依赖
mvn clean install -pl feign-noargs-demo -am

# 只编译 feign-noargs-demo 模块（不包括依赖）
mvn clean install -pl feign-noargs-demo
```

### 方式 2: 从模块目录编译

```bash
# 进入模块目录
cd springboot/feign-noargs-demo

# 编译本模块及子模块
mvn clean install
```

```bash

cd springboot/feign-noargs-demo/provider-service
mvn clean package

# 编译 consumer-service
cd springboot/feign-noargs-demo/consumer-service
```

## 常见问题

### Q1: 编译失败，提示找不到父 POM

**A**: 确保先编译父项目：

```bash
cd springboot
mvn clean install
```

### Q2: 依赖版本冲突

**A**: 本模块使用了不同的 Spring Boot 和 Spring Cloud 版本，这是正常的。模块 POM 中的 `dependencyManagement` 会覆盖父项目的版本。

### Q3: 启动脚本找不到 mvn 命令

**A**: 确保 Maven 已正确安装并配置到 PATH 环境变量：

```bash
# 检查 Maven 是否安装
mvn -version

# 如果未安装，请先安装 Maven
```

### Q4: 端口被占用

**A**: 使用停止脚本清理：

```bash
./停止脚本.sh
```

或手动查找并杀死进程：

```bash
# 查找占用端口的进程
lsof -i :8081
lsof -i :8082

# 杀死进程
kill -9 <PID>
```

## 与其他模块的关系

### 独立性

`feign-noargs-demo` 是一个独立的演示模块，不依赖其他子模块（如 `base-common`、`springboot-database-learning` 等）。

### 可移植性

如果需要将本模块独立出来：

1. 修改 `feign-noargs-demo/pom.xml`，将 parent 改为 `spring-boot-starter-parent`
2. 删除父项目 `pom.xml` 中的 `<module>feign-noargs-demo</module>`
3. 本模块即可独立运行
